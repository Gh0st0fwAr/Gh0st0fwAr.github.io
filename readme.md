## Окей, что мне с этим делать?
Чтобы развернуть проект, необходимо: 

    $ git clone git@github.com:Gh0st0fwAr/Gh0st0fwAr.github.io.git
    $ cd Gh0st0fwAr.github.io
    $ yarn

А также запустить скрипт:

    $npm run dev
    
Или 
    
    $yarn dev

## Что еще мне необходимо знать?
